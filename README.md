# Template: Standard Robot Framework

Simple example how to read data from QR code image file. Development file was created using [QR code generator](https://github.com/robocorp/example-qrcode-generator)

- Get started creating robots from a simple task template in `tasks.robot`.
  - Uses [Robot Framework](https://robocorp.com/docs/languages-and-frameworks/robot-framework/basics) syntax.
- You can configure your robot `robot.yaml`.
- You can configure dependencies in `conda.yaml`.
